컴퓨터 구조
1. 입력 받기 -> 2. 데이터 저장 -> 3. 데이터 처리 -> 4. 출력 정보 -> 5. 워크플로 제어

입력 장치
이 장치에는 컴퓨터에 데이터를 입력하는 데 도움이 되는 장치가 포함되어 있습니다. 이 장치는 사용자와 컴퓨터 사이에 링크를 생성합니다. 입력 장치는 정보를 컴퓨터가 이해할 수 있는 형식으로 변환합니다.

CPU(중앙처리장치)
CPU는 컴퓨터의 두뇌로 간주됩니다. CPU는 모든 유형의 데이터 처리 작업을 수행합니다. 데이터, 중간 결과, 명령어(프로그램)를 저장합니다. 컴퓨터의 모든 부분의 작동을 제어합니다.
CPU 자체에는 다음 세 가지 구성 요소가 있습니다.
ALU(산술 논리 장치)
메모리 유닛
제어 장치

출력 유닛
출력 장치는 컴퓨터에서 정보를 얻는 데 도움이 되는 장치로 구성됩니다. 이 장치는 컴퓨터와 사용자 사이의 링크입니다. 출력 장치는 컴퓨터의 출력을 사용자가 이해할 수 있는 형식으로 변환합니다.

컴퓨터 - CPU(중앙처리장치)
중앙 처리 장치(CPU)는 다음 기능으로 구성됩니다.
 - CPU는 컴퓨터의 두뇌로 간주됩니다.
 - CPU는 모든 유형의 데이터 처리 작업을 수행합니다.
 - 데이터, 중간 결과, 명령어(프로그램)를 저장합니다.
 - 컴퓨터의 모든 부분의 작동을 제어합니다.

메모리 또는 저장 장치
이 장치는 지침, 데이터 및 중간 결과를 저장할 수 있습니다. 이 장치는 필요할 때 컴퓨터의 다른 장치에 정보를 제공합니다. 내부 저장 장치, 주 메모리, 기본 저장소 또는 RAM(Random Access Memory)이라고도 합니다.
크기는 속도, 전력 및 성능에 영향을 미칩니다. 주 메모리와 보조 메모리는 컴퓨터의 두 가지 유형의 메모리입니다. 메모리 장치의 기능은 다음과 같습니다.
 - 처리에 필요한 모든 데이터와 지침을 저장합니다.
 - 처리의 중간 결과를 저장합니다.
 - 처리 결과가 출력 장치에 공개되기 전에 최종 처리 결과를 저장합니다.
 - 모든 입력과 출력은 메인 메모리를 통해 전송됩니다.

제어 장치
이 장치는 컴퓨터의 모든 부분의 작동을 제어하지만 실제 데이터 처리 작업을 수행하지는 않습니다.
본 장치의 기능은 다음과 같습니다.
 - 컴퓨터의 다른 장치 간 데이터 및 명령 전송을 제어하는 ​​역할을 담당합니다.
 - 컴퓨터의 모든 장치를 관리하고 조정합니다.
 - 이는 메모리에서 명령을 받아 이를 해석하고 컴퓨터 작동을 지시합니다.
 - 저장소의 데이터나 결과를 전송하기 위해 입출력 장치와 통신합니다.
 - 데이터를 처리하거나 저장하지 않습니다.

ALU(산술 논리 장치)
이 단위는 두 개의 하위 섹션으로 구성됩니다.
 - 산술 섹션
 - 로직 섹션

산술 섹션
산술 섹션의 기능은 덧셈, 뺄셈, 곱셈, 나눗셈과 같은 산술 연산을 수행하는 것입니다. 모든 복잡한 작업은 위의 작업을 반복적으로 사용하여 수행됩니다.

로직 섹션
논리부의 기능은 데이터의 비교, 선택, 일치, 병합과 같은 논리 작업을 수행하는 것입니다.
